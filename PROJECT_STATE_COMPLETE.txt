1. PROJECT OVERVIEW
================================================================================

Project Name: ImportationDotCA Inc.
Type: Static Next.js Website
Purpose: Wholesale import/export business website for décor, confectionery, 
         and hospitality goods
Target Market: Canada & USA
Base Color Scheme: Dark slate (#0b1220) with gold accent (#e3b341)
Design Style: Flat, modern, no rounded corners, no hover zooms

================================================================================
2. TECHNOLOGY STACK
================================================================================

- Next.js 14.2.33 (Static Site Generation)
- React 18.2.0
- TailwindCSS 3.4.0
- Swiper 11.0.0 (for image galleries)
- Framer Motion 10.16.0
- PHP (for contact form backend)

Build Configuration:
- Static export enabled (output: 'export')
- Images unoptimized for static hosting
- Trailing slash enabled
